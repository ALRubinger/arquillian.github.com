Arquillian Core

Arquillian Core Platform

Arquillian provides a component model for integration tests, which includes dependency injection and container life cycle management. Now, instead of having to manage a runtime in your test, Arquillian brings your test to the runtime. Arquillian Core, Container Extension + Protocols + Enrichers

Artifacts

Build dependency management artifact

Add the following stanza to the <dependencies> element within the <dependencyManagement> element in your pom.xml:

<dependency>
  <groupId>org.jboss.arquillian</groupId>
  <artifactId>arquillian-bom</artifactId>
  <type>pom</type>
  <version>1.0.0.CR7</version>
  <scope>import</scope>
</dependency>

Build dependency artifacts

Add the appropriate stanza to the <dependencies> element in your pom.xml:

JUnit Integration

<dependency>
  <groupId>org.jboss.arquillian.junit</groupId>
  <artifactId>arquillian-junit-container</artifactId>
  <scope>test</scope>
</dependency>

TestNG Integration

<dependency>
  <groupId>org.jboss.arquillian.testng</groupId>
  <artifactId>arquillian-testng-container</artifactId>
  <scope>test</scope>
</dependency>
