Arquillian Core 1.0.0.CR4 Released

Posted by Aslak Knutsen on Aug 16, 2011
Aug 16

The Arquillian team is proud to announce the 1.0.0.CR4 release of the Arquillian Core component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian Core
Version 1.0.0.CR4 view tag
Release date 2011-08-16
Released by Aslak Knutsen
Compiled against
  • ShrinkWrap Core – 1.0.0-beta-5
  • ShrinkWrap Descriptors – 1.1.0-alpha-2
  • TestNG – 5.14.6
  • JUnit – 4.8.1

Release notes and resolved issues 3

Quick fix CR4 release

Bug
  • ARQ-402 - Arquillian no longer works on Java 1.5
  • ARQ-549 - Guice is no longer a required dependency of TestNG, Arquillian TestNG Archive Appender should not force it

Thanks to the following list of contributors: Aslak Knutsen
