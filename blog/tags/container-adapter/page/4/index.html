Arquillian OSGi 1.0.0.CR4 Released

Posted by Aslak Knutsen on Aug 15, 2011
Aug 15
| Comments

The Arquillian team is proud to announce the 1.0.0.CR4 release of the Arquillian OSGi component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian OSGi
Modules
  • Arquillian OSGi Embedded Container Adapter
  • Arquillian OSGi Remote Container Adapter
Version 1.0.0.CR4 view tag
Release date 2011-08-15
Released by Aslak Knutsen
Compiled against
  • JUnit – 4.8.1
  • ShrinkWrap Core – 1.0.0-beta-5
  • Arquillian Core – 1.0.0.CR3

Release notes and resolved issues 0

Update to Core CR3

Thanks to the following list of contributors: Aslak Knutsen, Thomas Diesler
