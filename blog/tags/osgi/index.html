Arquillian OSGi 1.0.0.CR6 Released

Posted by Thomas Diesler on Jan 27, 2012
Jan 27
| Comments

The Arquillian team is proud to announce the 1.0.0.CR6 release of the Arquillian OSGi component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian OSGi
Modules
  • Arquillian OSGi Embedded Container Adapter
  • Arquillian OSGi Remote Container Adapter
Version 1.0.0.CR6 view tag
Release date 2012-01-27
Released by Thomas Diesler
Compiled against
  • Arquillian Core – 1.0.0.CR7
  • ShrinkWrap Core – 1.0.0-cr-1
  • JUnit – 4.8.1

Published artifacts

  • org.jboss.arquillian.container:arquillian-container-osgi-embedded:jar:1.0.0.CR6 [ jar | pom ]
  • org.jboss.arquillian.container:arquillian-container-osgi-remote:jar:1.0.0.CR6 [ jar | pom ]
  • org.jboss.arquillian.protocol:arquillian-protocol-osgi:jar:1.0.0.CR6 [ jar | pom ]
  • org.jboss.arquillian.testenricher:arquillian-testenricher-osgi:jar:1.0.0.CR6 [ jar | pom ]
