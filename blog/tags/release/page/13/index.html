Arquillian OSGi 1.0.0.CR3 Released

Posted by Thomas Diesler on Jul 25, 2011
Jul 25

The Arquillian team is proud to announce the 1.0.0.CR3 release of the Arquillian OSGi component!

What is Arquillian?

Arquillian is a platform that simplifies Java middleware testing. It brings your test to the runtime, freeing you from the concern of managing the runtime from your test. As a result, you can focus on the task at hand, writing your integration test logic.

Release details

Component Arquillian OSGi
Modules
  • Arquillian OSGi Embedded Container Adapter
  • Arquillian OSGi Remote Container Adapter
Version 1.0.0.CR3 view tag
Release date 2011-07-25
Released by Thomas Diesler
Compiled against
  • ShrinkWrap Core – 1.0.0-beta-3
  • Arquillian Core – 1.0.0.CR1
  • JUnit – 4.8.1

Release notes and resolved issues 1

Containers OSGi - Minor bug fix

Bug
  • ARQ-524 - Embedded OSGi container cannot find test class

Thanks to the following list of contributors: Thomas Diesler, Aslak Knutsen
